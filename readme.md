# has-dom

> Check if a JavaScript environment has a [DOM](https://en.wikipedia.org/wiki/Document_Object_Model)


## Install

```
$ npm install has-dom
```


## Usage

```js
const hasDom = require('has-dom');

// In Node.js
hasDom();
//=> false

// In the browser
hasDom();
//=> true
```
